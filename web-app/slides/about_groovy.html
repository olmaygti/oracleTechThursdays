What is Groovy?

Groovy is a dynamic language for the Java Platform:

  • Duck Typing
  • Dynamic Method Invokation Resolution (MOP)
  • Closures
  • Powerful scripting capabilities
  • Enhanced JDK (GDK)
  • DSL friendly
  • Introspection and metaprogramming made easy!
